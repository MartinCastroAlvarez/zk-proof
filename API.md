# Rust API Documentation

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/health` | GET | Check if the server is running |
| `/generate` | POST | Generate a proof |
| `/verify` | POST | Verify a proof |
| `/vk` | GET | Get the verifying key |
| `/auth` | POST | Set the credentials (address and secret key) |
| `/auth` | GET | Get the public address of the credentials |
| `/contract` | POST | Save the contract address |
